# Positive Vectors

Positive vectors contain inputs that must validate, encrypt/decrypt, derive,
verify, or round-trip successfully.

Each vector must identify:

- primitive or backend under test;
- suite or version identifier;
- input encoding;
- expected output or invariant;
- source of the vector when it comes from a standard, publication, or independent implementation.
